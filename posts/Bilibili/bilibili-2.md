## 基础快捷键

作为自启动核心配置，autoexec.cfg 承包了所有基础操作，无需手动加载，游戏启动自动生效：

Z、X、C、V 快速切换雷、闪、烟、火，滚轮上下都能跳，前侧键标点，后侧键直接开语音，投掷物切换不用再翻菜单

F5 到 F11 一键买雷、闪、烟、火、半甲、全甲、钳子，竞技模式抢时间必备，退格 Backspace 一键出售所有装备。

按 H 切换雷达缩放，N 键调整队友头顶标识，Delete 键一键消音到 30%，死斗必备。

按 - 开启 UI 隐藏，但保留击杀信息和准星，按 = 开关雷达显示。

平台按 K 一键发刀，5E 按 \ 一键报伤害

## 基于 autoexec 的其他 cfg 启动快捷键

快速切换不同 cfg 只需如下按键：

按 O 重启 autoexec；
按 P 开启一键跑图 practice；
按 `]` 开启看 demo 模式；
按 `[` 开启准星视角预设库；

## 其他控制台别名功能

临时开启功能，只需记住控制台代码：

输入颜色代码指令，更改默认 UI 颜色；

默认开启右键长按自动连镜，输入 debounce 关闭；

输入 silencer 禁用卸下消音器；
输入 avatars 显示对局头像；
输入 numbers 显示对局人数；
输入 tracer 关闭曳光弹；
输入 ping 开启快捷轮盘标点；
输入 squareon 强制方形小地图；
输入 squareoff 取消强制方形小地图；
输入 round 将计分板地图设置为圆形；
输入 sniperon 显示狙击镜扩散；
输入 sniperoff 关闭狙击镜扩散；
输入 ps 显示当前游戏延迟总结；
输入 pd 显示当前游戏延迟详情；
输入 show 常显装备栏；
输入 notshow 取消常显装备栏；
